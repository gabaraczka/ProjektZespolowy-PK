# Projekt zespołowy

STUDYBUDY to prosty system wspomagający pracę zespołu studentów – umożliwiający zarządzanie notatkami, generowanie podsumowań poprzez API oraz  śledzenia postępu prac projektowych za pomocą Jiry.

**Funkcjonalności**

-Tworzenie i wyświetlanie notatek projektowych

-Generowanie podsumowań pracy zespołu

-Logowanie i podgląd statusu zadań JIRA (integracja)

-Przyjazny interfejs użytkownika


**Technologie**

-HTML, CSS, JS (Frontend)

-PHP (Backend, REST API)

-SQLite/MySQL (baza danych)
